# Support

## Como obtener ayuda

- Revisa primero `docs/README.md`.
- Ejecuta `run.bat status` y comparte salida relevante.
- Si aplica, comparte codigo de error `SG-xxxx`.

## Antes de abrir issue

- Confirma que el problema sigue presente en la version actual.
- Adjunta entorno (Windows, Python, Node, npm).
- Describe pasos exactos para reproducir.
