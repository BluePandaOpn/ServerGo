# SG-0001 - Configuracion invalida

## Significado

La configuracion actual no cumple el formato esperado.

## Causas comunes

- `config.json` con campos invalidos o faltantes.
- Tipos de dato incorrectos (ej: texto en lugar de numero).

## Solucion

1. Revisar `config.json`.
2. Comparar con `docs/CONFIG.md`.
3. Corregir y reiniciar ServerGo.
