# SG-0002 - Dependencia faltante

## Significado

Falta una herramienta requerida para ejecutar una accion.

## Causas comunes

- `python`, `node` o `npm` no disponible en PATH.
- Dependencias del proyecto no instaladas.

## Solucion

1. Ejecutar `scripts/install.bat`.
2. Verificar `python --version`, `node --version`, `npm --version`.
3. Reintentar la accion.
