# SG-0003 - Operacion de red fallida

## Significado

La operacion no pudo completarse por un problema de red.

## Causas comunes

- Sin conexion a internet.
- Proxy o firewall bloqueando trafico.
- Timeout al consultar recurso remoto.

## Solucion

1. Verificar conectividad.
2. Revisar proxy/firewall.
3. Reintentar la operacion.
