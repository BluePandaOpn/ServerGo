# SG-0005 - Operacion de plugin fallida

## Significado

Fallo una accion relacionada con plugins.

## Causas comunes

- JSON de plugin invalido.
- Plugin con accion no soportada.
- Comando o URL del plugin mal definido.

## Solucion

1. Revisar archivos en `plugins/`.
2. Validar formato JSON.
3. Consultar `docs/PLUGINS.md`.
