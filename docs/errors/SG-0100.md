# SG-0100 - Fallo de arranque principal

## Significado

El proceso principal de ServerGo no pudo iniciar correctamente.

## Causas comunes

- Error en scripts de arranque.
- Dependencias incompletas.
- Configuracion incompatible.

## Accion recomendada

1. Ejecutar `scripts/doctor.bat`.
2. Ejecutar `scripts/install.bat`.
3. Revisar `.servergo/server.log`.
