# SG-0101 - Puerto en uso

## Significado

El puerto configurado ya esta ocupado por otro proceso.

## Causas comunes

- Otro servidor usando el mismo puerto.
- Proceso previo no detenido correctamente.

## Accion recomendada

1. Cambiar puerto en `Configuraciones`.
2. Detener proceso conflictivo.
3. Reintentar arranque.
