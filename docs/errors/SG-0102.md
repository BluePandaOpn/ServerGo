# SG-0102 - Estado/PID inconsistente

## Significado

El archivo PID no coincide con el estado real del proceso.

## Causas comunes

- Cierre forzado del servidor.
- PID stale en `.servergo/server.pid`.

## Accion recomendada

1. Detener manualmente el proceso si sigue activo.
2. Limpiar PID stale.
3. Iniciar nuevamente ServerGo.
