# SG-0200 - Configuracion corrupta

## Significado

`config.json` tiene estructura invalida o incompleta.

## Causas comunes

- JSON mal formado.
- Tipos invalidos en campos de configuracion.

## Accion recomendada

1. Validar formato JSON.
2. Restaurar defaults desde menu de configuracion.
3. Comparar con `docs/configuration/CONFIG.md`.
