# SG-0201 - Metadata de version invalida

## Significado

`Version.sv` no tiene campos esperados o contiene valores invalidos.

## Causas comunes

- Claves faltantes (`version`, `channel`).
- Formato incorrecto de lineas `clave=valor`.

## Accion recomendada

1. Revisar `Version.sv`.
2. Validar contra `docs/operations/VERSION_METADATA.md`.
3. Corregir y reiniciar.
