# SG-0300 - Dependencias Node incompletas

## Significado

Faltan paquetes Node requeridos por el servidor principal o APIs.

## Causas comunes

- `node_modules` ausente.
- `npm install` incompleto.

## Accion recomendada

1. Ejecutar `scripts/install.bat`.
2. Verificar `node` y `npm`.
3. Reintentar accion.
