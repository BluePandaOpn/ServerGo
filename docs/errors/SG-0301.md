# SG-0301 - Dependencias Python incompletas

## Significado

Faltan paquetes Python requeridos por ServerGo o proyectos generados.

## Causas comunes

- `.venv` no creado.
- `pip install -r requirements.txt` no ejecutado.

## Accion recomendada

1. Ejecutar `scripts/install.bat`.
2. Verificar entorno `.venv`.
3. Reintentar.
