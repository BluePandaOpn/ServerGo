# SG-0401 - Integridad de actualizacion fallida

## Significado

El paquete descargado no es confiable o esta incompleto.

## Causas comunes

- Descarga interrumpida.
- Archivo ZIP truncado/corrupto.

## Accion recomendada

1. Volver a publicar y descargar paquete.
2. Confirmar tamano/estructura del ZIP.
3. Reintentar instalacion.
