# SG-0500 - API Manager fallo de operacion

## Significado

El gestor de APIs no pudo completar una accion solicitada.

## Causas comunes

- Servicio mal configurado.
- Runtime faltante en una API.

## Accion recomendada

1. Revisar `api_services.json`.
2. Revisar logs y estado de APIs.
3. Corregir servicio y reintentar.
