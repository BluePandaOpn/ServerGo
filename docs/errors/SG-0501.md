# SG-0501 - API Manager conflicto de puertos

## Significado

Dos o mas APIs usan el mismo puerto.

## Causas comunes

- Alta manual con puerto repetido.
- Servicio externo ocupando el puerto destino.

## Accion recomendada

1. Asignar puertos unicos.
2. Detener procesos conflictivos.
3. Reiniciar servicios afectados.
