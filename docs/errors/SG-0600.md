# SG-0600 - Plugin con esquema invalido

## Significado

El plugin no cumple el esquema esperado por ServerGo.

## Causas comunes

- Campos obligatorios faltantes.
- Tipo de accion no soportado.

## Accion recomendada

1. Revisar el JSON del plugin.
2. Corregir formato y campos.
3. Consultar `docs/PLUGINS.md`.
