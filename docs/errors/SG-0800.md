# SG-0800 - Permisos insuficientes en filesystem

## Significado

El proceso no tiene permisos para leer/escribir archivos requeridos.

## Causas comunes

- Carpeta protegida.
- Archivo bloqueado por otro proceso.

## Accion recomendada

1. Cerrar procesos que bloqueen archivos.
2. Verificar permisos de la carpeta del proyecto.
3. Reintentar con privilegios adecuados.
