# SG-0900 - Recursos insuficientes

## Significado

El sistema no tiene recursos suficientes para completar la operacion.

## Causas comunes

- Memoria insuficiente.
- Disco lleno.
- Exceso de procesos simultaneos.

## Accion recomendada

1. Liberar memoria y disco.
2. Cerrar procesos no necesarios.
3. Reintentar operacion.
