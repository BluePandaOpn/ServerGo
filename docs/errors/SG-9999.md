# SG-9999 - Error no controlado

## Significado

Error generico no mapeado a una categoria especifica.

## Causas comunes

- Excepcion inesperada en tiempo de ejecucion.
- Estado inconsistente o corrupcion de datos.

## Solucion

1. Revisar `.servergo/server.log`.
2. Repetir pasos para identificar patron.
3. Reportar con contexto (accion, hora, mensaje exacto).
